# LLM & Neural Network Workshop

This repository contains Google Colab notebooks created while learning and practicing
core concepts of Neural Networks, Computer Vision, and Embeddings.

The notebooks are focused on hands-on implementation and understanding of architectures
commonly used in deep learning.

---

##  Notebooks Included

###  cnn_practice.ipynb
**Purpose:**  
Getting familiar with the **Convolutional Neural Network (CNN) architecture**.

**What it covers:**
- Basics of CNN
- Convolution layers
- Pooling layers
- Feature extraction from images
- Simple CNN model building

---

### 2️⃣ embedding.ipynb
**Purpose:**  
Understanding **vector embeddings** used in NLP and LLM-related tasks.

**What it covers:**
- What embeddings are
- How text is converted into vectors
- Basic embedding generation
- Use cases of embeddings in AI systems

---

### 3️ image_processing.ipynb
**Purpose:**  
Learning **image preprocessing techniques** before feeding data into neural networks.

**What it covers:**
- Image loading
- Image resizing and cropping
- Basic image manipulation
- Preparing images for CNN models

---

### 4️⃣ neural_network_using_pytorch.ipynb
**Purpose:**  
Building and training a **Neural Network using PyTorch**.

**What it covers:**
- Neural network fundamentals
- Model definition using PyTorch
- Training on the MNIST dataset
- Loss functions and optimization
- Model evaluation
